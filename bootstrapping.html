the way we solve this is by using the bootstrap. the usual assumptions are not met in this data set. in this case, well need to understand what the distribution of the intercept and slope are going to be under repeated sampling. well get that distribution using a bootstrap.

draw one bootstrap sample

boot_sample = function(df) {
  
  sample_frac(df, replace = TRUE) |> 
    arrange(x)
  
}

check if this works

boot_sample(sim_df_nonconst) |> 
  ggplot(aes(x = x, y = y)) +
  geom_point(alpha = 0.3) +
  stat_smooth(method = "lm") +

great! now, can I run my analysis on these?

boot_results = 
  boot_straps |> 
  mutate(
    models = map(.x = strap_sample, ~lm(y ~ x, data = .x)),
    results = map(models, broom::tidy)) |> 
  select(strap_number, results) |> 
  unnest(results)

what do I have now? lets look at the estimate of the sd under bootstrapping samples. what is the actual sd of the estimate under repeated sampling?
